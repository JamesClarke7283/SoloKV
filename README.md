# Solo KV Project

## Overview

Solo KV is a virtual workspace project comprising a key-value store library
(`libsolokv`) and a command-line interface (`solokv`). It aims to provide a
robust, extendable framework for working with key-value data across various
formats.

## Structure

- **`libsolokv`**: The core library supporting key-value operations, including
  support for multiple data formats like JSON and binary.
- **`solokv`**: A CLI tool leveraging `libsolokv` to perform data manipulation
  tasks from the command line.
